# 🚀 快速开始指南

## 第一步：安装依赖

打开命令行，进入项目目录，运行：

```bash
npm install
```

## 第二步：确认数据库

确保您的MySQL数据库已启动：
- 主机：localhost
- 端口：3307
- 用户名：root
- 密码：abcd1234
- 数据库：family_movie

## 第三步：启动服务器

```bash
node server.js
```

看到以下信息表示启动成功：
```
✅ 数据库连接成功！
✅ 服务器启动成功！
📡 服务器地址: http://localhost:3000
```

## 第四步：访问网站

打开浏览器访问：http://localhost:3000

## 第一次使用

1. **注册账号**
   - 访问 http://localhost:3000/login.html
   - 点击"注册"标签
   - 填写用户名和密码（邮箱和手机号可选）
   - 点击"注册"

2. **创建管理员账号**
   - 注册完成后，在数据库中执行：
   ```sql
   UPDATE user SET role = 'admin' WHERE username = '您的用户名';
   ```
   - 或者使用MySQL客户端工具修改user表的role字段为'admin'

3. **添加测试数据**
   - 先添加电影类型（在管理后台）
   - 然后添加电影（在管理后台）
